## Configuration

SkyrimPrefs.ini:

```
bSAO_CS_Enable=0
bSAOEnable=0
bEnableImprovedSnow=0
bVolumetricLightingEnable=1
bUse64bitsHDRRenderTarget=1
bDrawLandShadows=1
fGamma=1.0000
bDoDepthOfField=1
```

Skyrim.ini:

```
bVolumetricLightingDisableInterior=0
```
